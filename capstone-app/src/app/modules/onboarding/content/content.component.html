Learn the Basics of GMapr

GMapr Description Click to learn more about GMapr

GMapr provides a snapshot in history of cities from over seventy-five years ago.

 Maps Search Bar Learn more about how to use the Search bar

Use the search bar to search up any city in the world

 Map Custom Markers Learn more about map custom markers

Click any on the map to create custom markers. To remove custom marker either use the remove all custom marker button or right click the custom marker.

Map Type/Themes Learn more about Map Types and Themes

Use the Marker Types button to change the search marker type. Use the Map Theme button to change the map color theme.

Photo Filter Button Learn more about the Photo filter button

Use the filter button to display photos by marker type

 History Language Feature Click to learn more about Language Feature

Use the languages button to change the display language of the History section

 History Further Reading Links Click to learn more about Further Reading Links

Click on the further reading button to learn about history topic spoken about in the History section
